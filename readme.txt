Easily download other installed plugins as ZIP files from your WordPress admin interface.

== Description ==

Save Now allows WordPress administrators and developers to download installed plugins as ZIP files directly from the WordPress admin dashboard. This tool simplifies plugin backup and migration tasks, providing a convenient way to archive plugins for safekeeping or transfer between sites.

== Installation ==

1. Upload the `save-now` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit the Plugins page, where you'll now find a "Download" link next to each installed plugin.

== Frequently Asked Questions ==

= How does Save Now work? =

Save Now enhances the Plugins page in your WordPress admin area by adding a "Download" link next to each installed plugin. Clicking this link generates a ZIP file containing the plugin's files for easy downloading.

= What are the system requirements for Save Now? =

Save Now requires PHP version 7.2 or higher and WordPress version 5.0 or higher. Ensure that your server environment supports the ZipArchive PHP extension for proper functionality.

= Is Save Now secure? =

Yes, Save Now utilizes WordPress nonce verification and capability checks to ensure that only authorized administrators can download plugins.
